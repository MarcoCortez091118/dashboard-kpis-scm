## Manual Build 

> Install modules via `VENV`  

```bash
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
```

> Install Tailwind (another terminal)

```bash
$ cd static
$ yarn  
$ npx tailwindcss -i ./src/input.css -o ./dist/css/output.css --watch           
```

> Start APP

```
$ flask run 
```
